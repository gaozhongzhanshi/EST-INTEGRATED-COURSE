# AGENTS.md

## What this repo is

A single text file (`englishtest.txt`) containing English-Chinese translation exercises for a science & technology English course (科技英语综合教程). No code, no build system, no dependencies.

## Working with this repo

- The only file is `englishtest.txt` — a flat text file organized by Unit (1–7).
- Each unit has Chinese-to-English and English-to-Chinese vocabulary/phrase translation sections.
- Edits are plain text. No formatting conventions, no lint, no tests.
- Preserve the existing structure: `Unit N` headers, numbered items, `A.`/`B.` section labels.
